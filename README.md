# ORIS

ORIS = **Operational Reasoning & Integration System**

ORIS 不是聊天玩具，而是一个面向执行、具备推理能力、可整合多模型与多工具的 AI 研发员工系统。

一期目标：
- 不依赖聊天上下文
- 能读 GitHub 仓库
- 能遵守固定工程规则
- 改代码前先出 plan
- 修改后必须自检
- 能把记忆写回文件
- 能长期接续开发
